✨ Features

🧠 Machine Learning Prediction

An advanced AI model analyzes startup details such as funding, team size, founders, and investors to predict the probability of success.

📊 Analytics Dashboard

Provides a visual dashboard displaying important startup metrics, statistics, and performance insights in an easy-to-understand format.

📈 Success Probability Analysis

Calculates a success percentage score instead of a simple yes/no prediction, helping users better understand their startup's potential.

⚠️ Risk Assessment

Evaluates possible business risks and categorizes them into Low, Medium, or High Risk levels based on startup data.

📜 Prediction History

Stores prediction records in a CSV file, enabling tracking and analysis of previous startup evaluations.

⏱️ Real-Time Results

Generates instant predictions and analytics immediately after the user submits startup information.

🚀 User-Friendly Interface

Built with HTML, CSS, and Bootstrap to provide a clean, responsive, and easy-to-use experience.

🤖 Machine Learning Integration

Integrates a trained Random Forest machine learning model with Flask to provide intelligent predictions through a web application.
